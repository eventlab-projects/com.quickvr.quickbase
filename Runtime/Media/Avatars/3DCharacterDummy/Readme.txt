Thank you for downloading 3D Character Dummy.

There are two models included in the asset: Male Dummy and Female Dummy.

You can edit the models and then export them to Unity.

For Blender users, source files are included in 'SourceFiles' folder.

If your 3D software is not Blender you can try to import the .FBX file into your 3D software.
FBX files are in 'Models' folder.

Additional information like export options for Unity are included in the Blender files.
